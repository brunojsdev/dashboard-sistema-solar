# 🚀 Tech List - Gerenciador de Tarefas Futurista

Um gerenciador de tarefas (To-Do List) com design moderno inspirado em interfaces de ficção científica (Sci-Fi) e tecnologia, utilizando o conceito de *Glassmorphism* e animações fluidas em Canvas.

## 📋 Sobre o Projeto

O **Tech List** foi desenvolvido para oferecer uma experiência de usuário imersiva enquanto organiza as tarefas diárias. O projeto foca em alta performance visual, utilizando processamento gráfico via Canvas para o plano de fundo dinâmico e uma estrutura de código limpa e modular.

## ✨ Funcionalidades

* **Adição Dinâmica:** Adicione tarefas rapidamente pressionando "Enter" ou clicando no botão.
* **Gerenciamento de Estado:** Marque tarefas como concluídas ou remova-as da lista.
* **Contador em Tempo Real:** Visualização clara de quantas tarefas ainda restam no sistema.
* **Background Animado:** Efeito visual de partículas quadradas que reagem ao redimensionamento da tela.
* **Design Responsivo:** Adaptado para dispositivos móveis, tablets e desktops.
* **Feedback Visual:** Animações de entrada (*slide-in*) e efeitos de hover interativos.

## 🛠️ Tecnologias Utilizadas

* **HTML5:** Estruturação semântica do projeto.
* **CSS3:** * Variáveis CSS para fácil manutenção de cores.
  * Flexbox para layout responsivo.
  * Efeitos de *Blur* (Glassmorphism).
  * Animações via `@keyframes`.
* **JavaScript (Vanilla):**
  * Manipulação do DOM.
  * Lógica de banco de dados local (Array de objetos).
  * **HTML5 Canvas API:** Para a renderização das animações de fundo.
* **Google Fonts:** Utilização da fonte 'Space Grotesk' para um visual moderno.

## 📂 Estrutura de Arquivos

```bash

/
├── index.html    # Estrutura principal e links
├── style.css     # Estilização e temas
├── script.js     # Lógica de negócio e animações
├── README.md     # Documentação do projeto
└── img/          # Capturas de tela 
```
## 🚀 Como Executar o Projeto

1. Faça o download dos três arquivos (`index.html`, `style.css`, `script.js`).
2. Certifique-se de que todos os arquivos estão dentro da **mesma pasta**.
3. Abra o arquivo `index.html` em qualquer navegador moderno.
